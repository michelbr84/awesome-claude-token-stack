# Examples

Reference integrations for `awesome-claude-token-stack`. Each subdirectory is
self-contained and can be copied into your own project as a starting point.

## Available examples

- [`claude-code/`](./claude-code/) — Register `acts-mcp` with Claude Code,
  wire `acts observe stats` to the SessionStart hook, and enable the
  `acts-terse` skill.

## Coming in v0.2

- `cursor/` — `.cursor/mcp.json` integration.
- `gemini-cli/` — `~/.gemini/mcp_servers.json` integration.
- `codex/` — `~/.codex/config.toml` integration.
- `benchmark-harness/` — reproducible benchmark workload runner.
